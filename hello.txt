测试git命令
-----------------新建文件---------------------
添加到暂存区：git add 文件名
删除暂存区文件： git rm --cache 文件名
这个命令仅仅删除暂存区的文件，不会影响工作区的文件

------------------修改------------------------
查看日志 git reflog
查看详细日志 git log
------------------修改2----------------------
再次修改，再次提交
git add 文件名
git commit -m "提交信息"
git status :查看状态

------------------分支操作--------------------
新建hot-fix分支，并在hot-fix分支上进行操作
